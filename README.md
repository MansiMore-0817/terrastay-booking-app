## ğŸŒ Overview

**TerraStay** is a full-stack accommodation booking platform inspired by Airbnb.  
Users can browse listings, view details, upload multiple images using Cloudinary, and manage their own hosted properties.  
It is built following MVC architecture using **Node.js, Express, MongoDB, EJS, and Cloudinary**.
